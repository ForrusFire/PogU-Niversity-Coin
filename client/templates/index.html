Weepo Home

Hello! This is an experimental blockchain platform built by Sam Chun. This blockchain is secured with a Proof of Work mechanism (an alternate Proof of Stake consensus system is in progress), and it closely models the structure and architecture of Satoshi Nakomoto's original paper. The blocks currently store transaction data between addresses and they are linked together with SHA-1 hashes.

The REST API was built from Flask in Python and handles all requests to the blockchain, wallets, transactions, nodes, and mining. All endpoints are listed on the project's Github page.

In order to send and receive Weepo coins, you first need to create your RSA-secured blockchain wallet. The wallet's public key is your address, and the private key creates the signature that verifies transactions. Do not share your private key with anyone. There is a transaction fee of one Weepo Coin.

In order to mine blocks and receive rewards, you'll need to connect to the Weepo Blockchain port (default is 127.0.0.1:5000). Then head to 'Settings' and add the URL of at least one node that is already on the network; this will act as your "invite" to the main chain.
